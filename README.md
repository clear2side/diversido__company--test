# Тестовое задание от компании Diversido

> Сборка работает на gulp версии 4.0. 

#### Для начала работы

1. ```git clone https://github.com/clear2side/diversido__company--test.git```
2. ```cd diversido__company--test```
3. ```npm install gulpjs gulp-cli -g```  
4. ```npm install```
6. ```run gulp```
